What is Trackmania

Banner by Ubisoft Trackmania is a racing game licence containing 4 games all developped by nadéo and edited by Ubisoft on PC, Playstation 4, Xbox one, Wii and DS. The first Trackmania game was released in 2003. The goal of the game is very simple: drive to the finish the fastest way possible, whatever the way. The game developer are very lax, they don’t modify (or almost) the game and don’t correct the majority of bugs players find.
